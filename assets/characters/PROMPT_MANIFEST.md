# Maestro Combat 角色生成紀錄

## 共通身份母版

- 34 位音樂家先各自建立一張完整全身透明 PNG。
- 每位音樂家的三格姿勢表均以該張全身圖作為嚴格身份參考。
- 鎖定臉型、髮型、年齡、身形、服裝結構、服裝配色、鞋履、刺繡與代表道具。
- 統一風格：古典藝術幻想格鬥 RPG、日系角色主視覺、繪畫式賽璐璐明暗、精緻墨線、暖色輪廓光。
- 三格順序固定為：戰鬥、專屬大絕、非暴力 KO。
- 生成原稿使用純綠幕 `#00FF00`，再去背、等比縮放並輸出為 640×768 透明 WebP。

## 非暴力 KO 規格

角色只呈現精疲力竭、跪坐或低坐、道具安全放下與閉眼／暈眩神情；禁止受傷、疼痛、瘀青、血液、破損或武器。
