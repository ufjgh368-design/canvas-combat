# Maestro Combat 素材狀態

| 素材 | 狀態 | 說明 |
|---|---|---|
| 34 位音樂家圖鑑圖 | 已建立程式繪製的原創指揮家剪影 | 以角色色彩、姓名縮寫、音符與指揮棒辨識；不是史實肖像 |
| 戰鬥／終極／KO 影格 | 每位各三張獨立姿勢 WebP | 缺少史實角色圖時仍有完整、非暴力的演出，不顯示破圖 |
| 社群分享封面 | 已建立 Maestro Combat 專屬橫式主視覺 | 由內建圖像生成工具製作，含正確繁體中文標題與英文副標 |
| 場景 | 四時期 Canvas／CSS 舞台 | 巴洛克歌劇院、維也納音樂廳、浪漫風暴舞台、現代聲響劇場 |
| 音樂 | 既有 MP3 可選；Web Audio 合成為離線備援 | 缺少或播放失敗時不阻斷啟動 |

正式出版若需要歷史肖像級角色，可逐位以可靠早期肖像為參考重繪；目前版本已具備一致、可辨識且不冒充史實肖像的完整遊戲素材。
