# Maestro Combat 實作完成紀錄

## 2026-07-17 音樂家獨立版

以已驗證的 Canvas 對戰引擎為技術基底，建立獨立的「西洋音樂家知識格鬥 RPG」新專案與新網站，不覆蓋既有西洋藝術家網站。

- 保留七種模式、P1／P2／CPU 獨立狀態、可取消計時、鍵盤／滑鼠／觸控、無障礙、高 DPI Canvas、音效備援與版本化存檔。
- 37 位角色全面改為音樂家，依巴洛克、古典與過渡、浪漫與民族樂派、現代與跨界排列。
- 370 題涵蓋記憶、理解、素養應用、判斷、文化脈絡、趣聞、作品、國籍、時期與技法。
- 故事、試煉、Boss、教室、圖鑑、舞台、招式、文案與社群預覽全面音樂化。
- 舊藝術家存檔會安全遷移為音樂家新存檔；缺圖時以音符與樂器造型的 Canvas 戰鬥角色備援。
- 發布時建立新的 Sites 專案與網址，不沿用舊專案 ID。

## 現況檢查

專案維持無框架的 HTML／CSS／ES Modules／Canvas 2D 架構，保留既有互動概念，並將藝術家、題庫、關卡、AI、模式及存檔拆分為資料與模組。所有非同步戰鬥流程由 session scope 管理，返回選單或重新開始時會取消計時器與待執行工作。
