# 西洋音樂家知識格鬥：Maestro Combat

以繁體中文西洋音樂史問答驅動的 Canvas 2D 知識格鬥 RPG。玩家選擇作曲家，以作品、時期、曲式、和聲、配器與音樂史脈絡題發動招式。

## 內容

- 34 位音樂家，依巴洛克、古典與過渡、浪漫與民族樂派、現代與跨界四組排列。
- 340 題資料驅動題庫；簡單 114 題、中等 113 題、困難 113 題。
- 單人、雙人、大師挑戰、音樂試煉、練習、教室、音樂史旅程七種模式。
- P1、P2、CPU 獨立 HP、能量、Combo、分數與作答紀錄。
- 可取消的 CPU 行動與計時器；返回選單、暫停或頁面隱藏時安全停止。
- 鍵盤、滑鼠、觸控、高 DPI Canvas、降低動態／閃光、畫面震動與音量設定。
- 每位音樂家具圖鑑、戰鬥、終極招式與非暴力 KO 的原創指揮家剪影素材。

## 操作

- `Q` 主題動機、`W` 樂章重擊、`E` 終極交響。
- `1`–`4` 選擇答案，`Esc` 暫停並安全返回。
- 手機與平板可全程觸控操作。

## 驗證

```text
pnpm test
pnpm run build
```

正式發布前另執行技能稽核及桌機／手機瀏覽器流程測試。

## 主要資料

- `data/artists.js`：34 位音樂家、分組、招式、圖鑑與大師形態。
- `data/questions.js`：340 題題庫產生、驗證與選項洗牌。
- `data/story.js`：九章西洋音樂史旅程。
- `assets/ASSET_STATUS.md`：目前角色、封面、場景與音訊素材狀態。
- `data/question-sources.md`：內容來源類型與複核原則。
